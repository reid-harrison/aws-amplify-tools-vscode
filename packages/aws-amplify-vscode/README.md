# AWS Amplify VS Code Extension

<img src="https://s3.amazonaws.com/aws-mobile-hub-images/aws-amplify-logo.png" alt="AWS Amplify" width="550" >

Code snippets and completion for the [AWS Amplify](https://aws.github.io/aws-amplify) API.


## Installation

Go to the Extension Marketplace in VS Code and search for AWS Amplify Snippets.

## Usage

Start typing an AWS Amplify API command and choose the appropriate snippet. See the documentation for the complete set of prefixes and snippets.
